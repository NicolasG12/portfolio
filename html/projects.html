Space Shark

❮ ❯

Space Shark is a 3D Zero Gravity First Person Shooter made in Unreal Engine 5.

I programmed the procedural generation for the asteroid field in C++. I also worked on the object culling when it came to the asteroids to improve game performance. I also worked on the early versions of the design for the shark boss combat and getting the basic AI working. Apart from programming I was heavily involved with the modeling pipeline. I modeled and textured the solar panels, asteroids, and rocket launcher and then imported them into Unreal Engine. Another part of the game that I worked on was the sound, which was all done within Unreal Engine 5 using the MetaSounds framework.

Star-Force ULTRA!

❮ ❯

Star-Force ULTRA! is a 2D Turn Based RPG game prototype made using Unity.

I built and programmed the UI using Unity's UI builder while collaborating with the UI designers and communicating what was feasible in the UI builder.
I also worked on the combat system which follows a rock-paper-scissors style.
My primary role when working with the combat system was programming the game progression and making sure that the game would accurately react to scenarios such as a characters health reaching zero. Another aspect of the combat system that I worked on was making sure that the animations all played at the correct time.

Thimble Knight

❮ ❯

Thimble Knight is a 2D platformer made in Unity.

For this project I programmed the enemy behavior and the combat system in C#. In programming the combat I made sure that the animations lined up accurately to the actions of the player and the enemies using the animation tree tool in Unity. I also made sure that all the hitboxes were accurate and would interact correctly with all the varying physics systems in the game. When working on the combat system I also worked on the level design and level construction inside of Unity.

Coraline

❮ ❯

Coraline is a 3D horror puzzle game prototype based on the movie Coraline

I programmed the enemy movement in Unity using a waypoint system. Along with the enemy movement I also imported all the 3D assets into Unity from Blender. I worked closely with the art team on building the scene and making sure that the Unity engine lighting gave the desirable results.

Demons 101

First level of Demons 101
Hub level of Demons 101
❮ ❯

Demons 101 is a 2D web browser game made using JavaScript and the Phaser3 engine.

For Demons 101 I programmed the enemy movement, the progression system and the particles. I also worked in Tiled with the artist to design and create the levels.
The enemy path movement is also done in Tiled. A list of waypoints is generated from Tiled and the enemy moves towards them in order and reverse in the JavaScript code.
