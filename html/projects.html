Star-Force ULTRA!

❮ ❯

Star-Force ULTRA! is a 2D Turn Based RPG game made using Unity.

I built and programmed the UI using Unity's UI builder while collaborating with the UI designers and communicating what was feasible in the UI builder.

I also worked on the combat system which follows a rock-paper-scissors style. My primary role when working with the combat system was programming the game progression and making sure that the game would accurately react to scenarios such as a characters health reaching zero.
